Overview

This module introduces the ESP32 microcontroller and basic electrical circuit concepts including resistors, LEDs, pushbuttons, and breadboard wiring.

What You’ll Learn

  • ESP32 pin configuration and features
  • Powering the ESP32
  • Basic input/output: blinking LEDs, reading pushbuttons
  • Wiring circuits on a breadboard
